ineter universidad nacional ingeniera uni ambos managua nicaragua abstract gis based software is presented which permit the estimation of seismic vulnerability and the presentation of results in digital maps for single hoús groups of buildings parts of settlements or evir complete towns nicaragua is country with high seismic activity thus seismic vulnerability and risk studi have been carried out in recent years by several scientific groups applying different methodologi the assessment of seismic vulnerability requir the execution of distinct tasks recollection of field data integration of datar from the municipal cadastre reprocessing or screening to test the reliability of the data definition of calculation of vulnerability functions calculation of vulnerability for single objects as houses or buildings calculation of mean vulnerability for certain areas as barrio or squar finally the results are normally presented on maps observing the work flows of the projects carried out in nicaragua we can see that the preparation of datar and the presentation of the results take much time mainly because several separate software tools are used for distinct tasks and scientific calculations thir the results have to be passed to other softwarar used for the graphical presentation of results it is common that all the process has to be donir repeatedly to observar the influence of different parameters to the result in order to reducir time and effort to be spent with several unspecialized tools and procedur an integrated software system was created the user of which has not to care about separate software tools for each part of the process the main advantage of the software is the combination of geographical information system gis with the logics that surrounds the specific methodologi of seismic vulnerability index index of damages and presentation of results this advantage is accomplished by exploiting the ability of customizing the gis software arcgis esri trough of its several software librari with the specialized functionalities and the logical programming with the requested user requirement the new software uses connection with an external centralized enterprise data base which stores all the input information and calculation results and which is automatically synchronized for the presentation of results using gis when new requirements of typologies for houses are introduced cadastral datar from the municipality is uploaded to the system this information becomes the base data set of the current project the cadastral information contains datar on the constructive type of the housar dimensions year of construction type of walls roof number of inhabitants etc these datar are then processed to define groups of houses belonging to certain types or class for each type number of houses is selected which will be visited to confirm the correctness of the classification the other not selected hoús get the average value of its type this process can be run repeatedly to generate distinct results for several set of input parameters visualizing the results automatically over map layout the system also allows to present damage scenario for specific seismic events with givir hypocenter and magnitude moreover there is another and just as important byproduct the documentation of the software serv as guide for students working on object oriented softwarar engineering by using unified modeling language uml and software logic architecture this work was carried out at the geophysical department of the instituto nicaragense
